1. Project name and purpose
   - Project Name: Hamburgerkuppen
   - Purpose: Provide an example of a social engineering attack (Phishing specifically) and information about phishing attacks.

2. Scope
    - This project focuses on educating users about phishing attacks through a simulated example and informational content.
    - The main components include a phishing simulation and educational materials through corresponding government and technical websites.

3. Ethical notice
    - This project is intended for educational purposes only.
    - It aims to raise awareness about phishing attacks and promote cybersecurity best practices.
    - Users are advised to use the information responsibly and ethically.
    - The project does not endorse or encourage any malicious activities and is not intended to be used for harmful purposes.

4. Quick start guide
    - To get started with the project, follow these steps:
      1. Clone the repository to your local machine.
      2. Review the phishing simulation example provided in the project report.
      3. Explore the educational materials and resources linked within the project.
      4. Share the knowledge gained with others to promote awareness about phishing attacks.
